Entropy


  1. It's an app designed to control a device using any other device connected over any local network.

  2. The device on which entropy is installed can be operated through any other device.

  3. Entropy generates a QR code and a link that can be used to access that specific device.

  4. Device running entropy becomes the host.
  5. You can perform various functions from starting apps to closing an already running app or taking screenshots to shutting down the pc itself.
  6. The link opens a web page on the other device from which you can control the host device.
